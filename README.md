# 🍫 Interactive Recipe Card with Animations

An interactive and responsive Recipe Card built using **HTML, CSS, and JavaScript**. This project showcases a delicious recipe with engaging animations, smooth transitions, and interactive cooking steps. Users can view ingredients, follow step-by-step instructions, track progress, and enjoy a visually appealing cooking experience across all devices.

## 🚀 Features

### 📋 Recipe Information
- Recipe title and featured dish image
- Preparation time and serving size
- Ingredients list
- Step-by-step cooking instructions

### 🎨 CSS Styling & Animations
- Modern and responsive card design
- Hover effects on ingredients and instructions
- Smooth transitions for content visibility
- Animated progress bar for cooking progress
- Attractive buttons and visual elements
- Consistent typography, spacing, and color scheme

### ⚡ JavaScript Interactivity
- Show/Hide Ingredients functionality
- Show/Hide Instructions functionality
- "Start Cooking" feature to begin guided cooking mode
- "Next Step" button to navigate through instructions sequentially
- Dynamic step highlighting during cooking

### 📱 Responsive Design
- Mobile-friendly layout
- Tablet and desktop optimization
- Flexible image and content scaling
- Media queries for improved user experience across devices

### ⭐ Bonus Features
- Cooking timer that starts when cooking begins
- Print-friendly recipe layout using CSS media queries
